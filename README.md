# API testing

This is an api-testing repo for https://github.com/filok94/api_hand

## Description

https://github.com/filok94/api_hand is node.js nest project with mongodb as database

## Getting Started

### Dependencies

* playwright 1.27.1
* mongodb 4.11

### Executing program

* To start test add .env file
* Install dependencies
*
```
npm i
```
*
```
npm run test
```
